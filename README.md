## Overview

This portfolio is not another template-cloned landing page stuffed with generic gradients and meaningless animations.

It was designed as a system-driven personal space:
- clean architecture
- modular frontend structure
- interactive transitions
- immersive UI behavior
- performance-oriented rendering
- aesthetic but readable design language

The goal was simple:

> Build something that feels engineered, not assembled.
